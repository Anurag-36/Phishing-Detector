# Release Notes

## v0.1.0 - Test Version

This is the first test version of Phishing Detector.

### What’s Included

- Rule-based phishing detection for URLs and page content
- Optional reputation checks through the local proxy server
- Clear risk scores and short explanations
- Local-only handling for sensitive API values

### In Progress

I am actively working on improvements and additional features, including better detection logic, UI refinements, and expanded reputation checks.

### Notes

- This release is intended for testing and review.
- Secrets and API keys should stay in local `.env` files only.
